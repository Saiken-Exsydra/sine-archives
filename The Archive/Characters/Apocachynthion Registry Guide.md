# Apocachynthion Registry Guide

Short reference for filling the `Apocachynthion Registry` section in character dossiers and entry files.

## Standard Template

```md
## Apocachynthion Registry

**File Type:** ...
**Registry Status:** ...
**Rank:** ...
**Mark:** ...
**Open Mark:** ...
**Margins:** ...
**Restrictions:** ...
**Point:** ...
**CAS Access:** ...
**Leaf Permissions:** ...
**Point Register:** ...
**Anchor:** ...
**Anchor Reading:** ...
**Anchor Depth:** ...
**Reach:** ...
**Overreach History:** ...
**Apparatus Permissions:** ...
**Registration Tenure:** ...
**Academy Record:** ...
**Practice Record:** ...
**Blots:** ...
**Pager Note:** ...
**Current Seal:** ...
```

## Allowed Values

**File Type**
- `Practice File`
- `Sealed File`
- `House-restricted file`

**Registry Status**
- Usually one of:
- `Registered`
- `Registered under [compact]`
- `No ordinary Apocachynthion practitioner file`

**Rank**
- `Nib`
- `Quill`
- `Stylus`
- `Archtype`

**Mark**
- `None`
- `Mark 1`
- `Mark 2`
- `Mark 3`
- `Mark 4`
- `Mark 5`
- Use the highest confirmed Mark on file.

**Open Mark**
- Usually:
- `None`
- Temporary or emergency expansion may be recorded as:
- `Open Mark: Mark [number], [condition / operation / theater / seal]`
- Open Mark is permission, not power.

**Margins**
- Use standardized categories, not character-specific sentence descriptions.
- Approved examples:
- `Civic`
- `Military`
- `Restricted`
- `Instructional`

**Restrictions**
- Use short institutional phrasing.
- Examples:
- `No martial apparati`
- `No independent field work`
- `No thin-site practice without supplemental seal`
- `Restricted Reach testing only`

**Point**
- Use short status phrasing.
- Examples:
- `Academy-issued; carried`
- `Academy-issued; implanted`
- `Inherited core integrated`
- `Not applicable`

**CAS Access**
- Common values:
- `Basic CAS`
- `Basic CAS under supervision`
- `Professional CAS under license`
- `Restricted CAS locked`
- `Not applicable`

**Leaf Permissions**
- Common values:
- `Common Leaves`
- `Common Leaves under supervision`
- `Licensed Leaves by Mark permission`
- `Restricted Leaves by explicit seal only`
- `None`
- `Not applicable`

**Point Register**
- Keep legal and technical, not psychological.
- Examples:
- `Protected; no open review`
- `Protected; specialist review window sealed`
- `Protected; lawful access required`
- `Not applicable`

**Anchor**
- Character-specific.
- Keep it simple: one word or a short pair of words only.

**Anchor Reading**
- Usually:
- `Confirmed`
- `Disputed`
- `Unassessed`

**Anchor Depth**
- Use only:
- `Shallow Anchor`
- `Deep Anchor`
- `Abyssal Anchor`
- If necessary for split cases:
- `Shallow Anchor / Deep Anchor`
- `Deep Anchor / Abyssal Anchor`
- `Abyssal review pending`

**Reach**
- Use only:
- `Bound Reach`
- `Near Reach`
- `Working Reach`
- `Wide Reach`
- `Unmoored Reach`
- If unassessed:
- `Unmeasured`

**Overreach History**
- Keep short.
- If none:
- `None recorded`
- If relevant:
- `Assessment symptoms recorded`
- `Repeated Overreach under review`
- `Anchor Bruising monitored`
- `Anchor Fray concern`

**Apparatus Permissions**
- Use the apparatus categories from the Redactory Codex.
- Common values:
- `Ordinary Apparati`
- `Professional Apparati under license`
- `Martial Apparati under military Margin`
- `Restricted Apparati by explicit seal only`
- `No apparati outside supervised training`

**Registration Tenure**
- Format as either:
- `[number] years`
- `Undisclosed`

**Academy Record**
- One short institutional line.

**Practice Record**
- One short institutional line.

**Blots**
- Keep short.
- If none:
- `None`

**Pager Note**
- Usually:
- `None`
- `Review`

**Current Seal**
- One short institutional line.

## Usage Notes

- Keep the registry sterile and standardized.
- Do not explain the character here beyond the institutional record.
- Do not turn `Margins`, `Anchor Depth`, `Reach`, `Restrictions`, or `Apparatus Permissions` into prose.
- Keep `Anchor` short even when the lore behind it is complicated.
- Rank measures training and discipline.
- Mark measures authorization.
- Anchor Depth measures vertical stratal position.
- Reach measures lateral operational flexibility from the Anchor.
- Point registers record routing activity, not motives or private thoughts.
- Leaf means a BrightCrystal-inscribed routing schema, not a registry file.
