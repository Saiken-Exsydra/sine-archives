# Apocachynthion Registry Guide

Short reference for filling the `Apocachynthion Registry` section in character dossiers and entry files.

## Standard Template

```md
## Apocachynthion Registry

**Leaf Type:** ...
**Registry Status:** ...
**Rank:** ...
**Mark:** ...
**Open Mark:** ...
**Margins:** ...
**Anchor:** ...
**Anchor Reading:** ...
**Depth Reading:** ...
**Reach Reading:** ...
**Registration Tenure:** ...
**Academy Record:** ...
**Practice Record:** ...
**Blots:** ...
**Pager Note:** ...
**Current Seal:** ...
```

## Allowed Values

**Leaf Type**
- `Practice Leaf`
- `Sealed Leaf`
- `House-restricted leaf`

**Registry Status**
- Usually one of:
- `Registered`
- `Registered under [compact]`
- `No ordinary Apocachynthion practitioner file`

**Rank**
- `Nib`
- `Quill`
- `Stylus`
- `Archtype`

**Mark**
- `None`
- `Mark 1`
- `Mark 2`
- `Mark 3`
- `Mark 4`
- `Mark 5`
- Use the highest confirmed Mark on file.

**Open Mark**
- Usually:
- `None`

**Margins**
- Use standardized categories, not character-specific sentence descriptions.
- Approved examples:
- `Civic`
- `Military`
- `Restricted`
- `Instructional`

**Anchor**
- Character-specific.
- Keep it simple: one word or a short pair of words only.

**Anchor Reading**
- Usually:
- `Confirmed`
- `Disputed`
- `Unassessed`

**Depth Reading**
- Use only:
- `Shallow`
- `Deep`
- `Abyssal`
- If necessary for split cases:
- `Shallow / Deep`
- `Deep / Abyssal`

**Reach Reading**
- Use only:
- `Minimal`
- `Broad`
- `Catastrophic`

**Registration Tenure**
- Format as either:
- `[number] years`
- `Undisclosed`

**Academy Record**
- One short institutional line.

**Practice Record**
- One short institutional line.

**Blots**
- Keep short.
- If none:
- `None`

**Pager Note**
- Usually:
- `None`
- `Review`

**Current Seal**
- One short institutional line.

## Usage Notes

- Keep the registry sterile and standardized.
- Do not explain the character here beyond the institutional record.
- Do not turn `Margins`, `Depth Reading`, or `Reach Reading` into prose.
- Keep `Anchor` short even when the lore behind it is complicated.
