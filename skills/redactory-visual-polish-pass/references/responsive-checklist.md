# Responsive Checklist

Check at desktop, tablet, and mobile widths.

## Layout

- No horizontal overflow.
- Text does not overlap controls or images.
- Sticky elements either behave correctly or become static on mobile.
- Full-screen experiences respect viewport height and safe areas.
- Grid layouts collapse to one column where needed.

## Text

- Long words, labels, and buttons fit containers.
- Body text stays readable.
- Metadata remains useful, not cluttered.
- Decorative text does not crowd primary content.

## Interaction

- Touch targets are large enough.
- Hover-only content has focus/touch equivalent.
- Modals/panels can be dismissed.
- Scrollable panes are discoverable.

## Media

- Images keep stable aspect ratios.
- Important image subjects remain visible.
- Background art does not hide text.
- Loading/fallback states do not shift layout badly.
