# Lore Page Structure

Use the structure that matches the content.

## Character Dossier

- portrait or hero rail.
- quote or registry note.
- file fields.
- main dossier body.
- ability or status fragments.
- archive note.
- related entries.

## Classified Document

- classification header.
- metadata strip.
- warning or access note.
- sectioned body.
- annotations or marginalia.
- addendum blocks.
- related files.

## Redactory Theory Or Index Page

- concept title and operational summary.
- diagram, theorem, or object surface when available.
- definitions.
- procedural or doctrinal sections.
- warnings/limits.
- cross-links to systems, Redactory, Divination, Shores, or Index material.

## Dark Ends Entry

- avoid generic horror-blog framing.
- use restraint, archival distance, file evidence, timestamps, place/person references, and incomplete fragments.
- preserve readability and canon boundaries.
